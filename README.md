# Kathmandu Air Pollution & Respiratory Health Dashboard

🚧 **Status: In Progress**

An interactive data science dashboard analyzing air pollution trends in Kathmandu and their impact on respiratory health.

---

## 🔍 Features

* PM2.5 trend analysis with WHO thresholds
* Seasonal pollution patterns
* Health risk scoring
* Lag correlation (pollution → symptoms)
* Forecasting (SARIMAX, currently placeholder data)

---

## 📊 Data Sources

* Open-Meteo API
* Google Trends
